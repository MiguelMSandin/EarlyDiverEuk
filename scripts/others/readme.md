# Others/
  
In this folder there are a collection of scripts that contributed to this study, but that are not part of the analyses.
  
[phylopic.R](https://github.com/MiguelMSandin/EukEcoEvo/tree/main/scripts/others/phylopic.R): This script was used to download the silhouettes from [phylopic.org](https://www.phylopic.org/) used in Fig. 1.  
  
[envVSdes.R](https://github.com/MiguelMSandin/EukEcoEvo/blob/main/scripts/others/envVSdes.R); This script will help you identify 'environmental' or 'described' sequences shown in **Supplementary Material Fig S8**. Feel free to modify the regular expressions to please your needs and see what happens.
